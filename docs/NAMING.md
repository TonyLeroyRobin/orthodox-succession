# Naming

> **PLACEHOLDER** (ROADMAP_ADDENDUM §B2). To be drafted **with the maintainer
> in chat**, not generated unilaterally.

The stance already binding on every record:

**Historical names in historical contexts, modern official names for modern
entities, all variants stored and searchable.** Person display names carry no
honorific prefix (office is expressed by the Tenure); native scripts and
transliteration variants live in `names.native` / `names.variants`.
Transliteration conventions: Greek ALA-LC; Russian/Slavonic simplified
BGN/PCGN; Georgian national; Arabic ALA-LC (CLAUDE.md).
