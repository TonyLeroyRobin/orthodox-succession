# Repositories — the controlled list for edition links

*P4 (PERSONS_LIBRARY_CONTROVERSIES). Edition URLs may point only at the
repositories below; validate.py warns on any other host, and extending this
list requires maintainer approval. The standing rule: LINK, NEVER RE-HOST.*

## Public-domain text repositories

- Christian Classics Ethereal Library — host: `ccel.org`
- New Advent — host: `newadvent.org`
- The Tertullian Project (incl. its additional-translations collection) — host: `tertullian.org`
- Early Christian Writings — host: `earlychristianwritings.com`
- Documenta Catholica Omnia (PG/PL scans — PG column citations link to page images where feasible) — host: `documentacatholicaomnia.eu`
- Patristica.net (PG/PL indexes) — host: `patristica.net`
- Internet Archive — host: `archive.org`
- HathiTrust — host: `hathitrust.org`
- Syriaca / syri.ac (Syriac transmission) — host: `syri.ac`

## In-copyright series (link to publisher or WorldCat only)

- WorldCat — host: `worldcat.org`
- WorldCat search — host: `search.worldcat.org`
(Sources Chrétiennes, Fathers of the Church, Popular Patristics/SVS,
Dumbarton Oaks Medieval Library and similar series link via the hosts above
per the existing rights rules.)

## Adding a repository

Propose the host with its rights rationale; the maintainer approves by name.
The validator reads the `host:` back-ticked values on this page.
