# Contributing

> **PLACEHOLDER** (ROADMAP_ADDENDUM §B2). The full contribution guide is to be
> drafted **with the maintainer in chat**, not generated unilaterally. Until
> then, the binding rules are the short version below plus
> [`CLAUDE.md`](CLAUDE.md) and [`KICKOFF.md`](KICKOFF.md).

The short version, already in force:

- **Corrections come as pull requests with graded sources.** Every substantive
  claim needs a citation graded `primary | official-list | scholarly |
  tradition`; web sources need an `archived_url`.
- **Disputes are resolved by sourcing, not argument.** Where reputable lists
  disagree, the record carries both witnesses (`disputed` status or
  per-recognizer `recognition` entries) — the database records, it does not
  adjudicate.
- **Validation gates merges.** `python scripts/validate.py` must exit 0.
- **IDs are permanent** from v0.1.0 onward. Never rename a published ID.
- No contribution channel exists outside GitHub pull requests (standing
  boundary, ROADMAP_ADDENDUM §D).
